# chat-room

## Technical Stack

- React.js
- Socket.io
- Material UI


## About Project

- This is chatting app. More than two peoples are chatting in a group using room.

## How to install

- Download or Clone the repo
- Go to the client folder or run this command
```
cd client
```
- Run this command
```
npm install
```
then

```
npm start
```
- Go to the server folder or run this command
```
cd server
```
then
```
npm install
```
then
```
npm start
```
or
```
npm run dev
```
